## ğŸš€ Quick Start

### Prerequisites
- Node.js (v12 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/abdullahjave/Snake-game-Modern-JavaScript.git
   cd Snake-game-Modern-JavaScript
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the server**
   ```bash
   npm start
   ```

4. **Open your browser**
   ```
   http://localhost:3000
   ```
